# Security Policy

## Reporting a Vulnerability

**Email:** security@echoself.ai
**Response time:** < 48 hours acknowledgment, < 7 days severity assessment

Please do NOT report security vulnerabilities through public GitHub issues.

## Scope

- ECHO//SELF mobile applications (iOS and Android)
- Supabase edge functions and API endpoints
- User authentication and data storage

## Out of Scope

- Theoretical vulnerabilities without proof of concept
- Third-party services (Supabase, OpenAI, Stripe — report to them directly)

## Responsible Disclosure

We follow a 90-day coordinated disclosure policy. Researchers who responsibly disclose vulnerabilities are credited in our Hall of Fame.
